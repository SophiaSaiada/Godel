# תחביר

בעמוד הזה יוגדר תחביר השפה בצורה מדוייקת באמצעות הפורמט EBNF.

  • # Letter

    • [a..z]
    • [A..Z]

  • # Digit

    • [0..9]

  • # Underscore

    • _

  • # simpleName

    • Underscore
    • Underscore? Letter (Letter | Digit | Underscore)*

  • # type

    • <any type defined in the code>

  • # DecimalLiteral

    • Digit+

  • # FloatLiteral

    • DecimalLiteral . DecimalLiteral

  • # StringLiteral

    • \" <any UTF-8 character other than quote or newline> \"

  • # BooleanLiteral

    • true
    • false

  • # SEMI

    • ;
    • \n

  • # expression

    • BooleanLiteral
    • DecimalLiteral
    • FloatLiteral
    • StringLiteral
    • functionCall
    • block

  • # value

    • val simpleName : type = expression

  • # declaration

    • function
    • value

  • # statement

    • declaration
    • expression

  • # block

    • { SEMI* statement (SEMI+ statement)* SEMI* }

  • # functionParameter

    • simpleName : Type

  • # function

    • fun
      (Type .)?
      simpleName
      (< Type >)?
      ( functionParameter* )
      : Type
      functionBody

  • # functionBody

    • = expression
    • block

  • # functionCall

    • simpleName ( expression* ) block?
